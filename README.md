# 每日音乐移动端交互 Demo

一个面向移动端音乐产品的前端交互 Demo，重点展示移动端 UI 设计、页面组织、播放器交互和产品化细节。项目参考成熟音乐 App 的信息架构，但在视觉语言、模块内容和交互动效上做了差异化设计。

## 在线预览

部署后可填写：

```text
https://your-demo-url.example.com
```

## 项目亮点

- 首页频道：支持推荐、声场、播客、听书、夜场等内容区切换，并带有滑动动效。
- 播放器：实现迷你播放器、全屏播放器、胶片旋转、歌名横向循环滚动、播放队列和音效调节面板。
- 好友模块：包含好友动态、一起听房间、好友筛选、在线状态和朋友在听内容。
- 灵感笔记：展示音乐笔记、创建、收藏等内容入口。
- 我的页面：支持用户资料卡、头像、用户名、签名、音乐号、背景卡主题等个性化展示。
- 移动端适配：围绕手机视口进行布局、字号、卡片密度和交互区域优化。

## 技术栈

- Vue 3
- TypeScript
- Vite
- CSS3 动效与响应式布局

## 本地运行

```powershell
cd frontend
npm install
npm run dev
```

浏览器打开：

```text
http://localhost:5173
```

## 构建

```powershell
cd frontend
npm run build
```

## 项目定位

这是一个前端交互与移动端 UI 作品 Demo，主要用于展示产品界面设计、交互动效、组件组织和响应式实现能力。当前未接入真实音乐版权、用户登录和后端数据服务。
